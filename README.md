# OMNIEVENTS NEXT.js WEB APPLICATION

## SPECS:

* Frontend: Next.js
* Backend: Node.js V20.20.1
* DB: PostgreSQL 17

## DISCLAIMER:

### * This app is for demo purposes only! This is an MVP project.
### * The backend of this app is fully live on Render.
#### * And because I'm using a free tier, at first login, there is a delay [ Cold Start ] of a few seconds!


## INSTALLATION (Locally):
1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
